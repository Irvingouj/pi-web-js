QuickJS ephemeral-VM bench — @pi-oxide/web-js

ready — click "Run bench".

Measures the per-handler cost of new WebSession() → runCellAsync(code) → free() — a fresh QuickJS Runtime+Context per event, deallocated after. The bootstrap line is the one-off module decode+compile+instantiate (Phase A). init = Runtime+Context + bindings (Phase B+C). Memory is the wasm linear heap; a plateau means the Rust allocator is reusing its pool across handlers — the core argument for the ephemeral model vs a persistent VM.
